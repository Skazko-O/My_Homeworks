Елемент <input>

Одним з найбільш використовуваних елементів форми

Основні атрибути:

  • type – визначає тип введення (текст, число, дата і т. д.).
  • name – задає ім'я, яке використовується для надсилання даних форми.
  • value – встановлює початкове значення поля.
  • placeholder – відображає текст-підказку.
  • required – робить поле обов’язковим для заповнення.
  • disabled – вимикає поле, роблячи його недоступним.
  • readonly – робить поле доступним для перегляду, але не для редагування.

Елемент <lable>

Використовується для зв’язку текстового опису з елементами форми, такими як <input>, <textarea>, <select>

Корисні атрибути:

  • for – вказує id елемента, до якого прив’язаний label.
  • title – додає підказку при наведенні.

Елемент <select>, <option> і <optgroup>

Використовується для створення випадаючого списку, з якого користувач може вибрати один або кілька варіантів. Елемент option визначає окремі варіанти вибору.

Атрибути:

  • name – використовується для передачі даних у формі.
  • multiple – дозволяє вибрати кілька варіантів.
  • size – задає кількість видимих пунктів списку.
  • disabled – блокує вибір.
  • required – робить вибір обов'язковим.

Елемент <textarea> разом з <button>

Використовується для введення великого обсягу тексту, наприклад, коментарів або описів

Корисні атрибути:

  • name – передає дані у формі.
  • maxlength – обмежує кількість символів.
  • readonly – тільки для читання.
  • wrap – визначає, як текст переноситься
  • required – робить вибір обов'язковим.
  • disabled – вимикає поле вводу.

Елементи <fieldset> і <legend>

Використовуються для групування та заголовку частин форми, що покращує її структуру та доступність. Елемент legend встановлюэ заголовок.

Елемент <datalist>

Дозволяє створювати список автозаповнення для <input>. Він допомагає користувачам вибирати значення зі списку, зберігаючи можливість вводити власний текст.
